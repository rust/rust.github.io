ruby-trunkからのbuild

2008/07/28

先っちょを使うと宣言した以上，trunkを使わねばと言うことで，そのログなどを．

  • Subversion Repository から checkout
% mkdir ruby-repos
% cd ruby-repos
% svn co http://svn.ruby-lang.org/repos/ruby/trunk/ ruby-trunk
  • http://doc.loveruby.net/wiki/HowToInstallRubys.htmlを参考に，/usr/local/ruby-pkg/ruby-1.9-trunk にインストール．
% cd ruby-trunk
% autoconf
% ./configure --prefix=/usr/local/ruby-pkg/ruby-1.9-trunk
% make
  • と，ここで，bisonがないと言われるので持ってくる1．
% apt-get install bison

% sudo make install
% sudo ln -s /usr/local/ruby-pkg/ruby-1.9-trunk/bin/ruby /usr/local/bin/ruby1.9
  • 確認
% rehash
% ruby1.9 --version
ruby 1.9.0 (2008-07-28 revision 18249) [i686-linux]
  1. まあ当然必要だよねorz [return]
