##Debian で Ocaml を使ってみた

  • 関数型言語
  • advi とか unison とかパッケージが ocaml で書かれているらしい
  • Tutorial がいいらしい
  • 型推論がない！
    • + 演算子は int -> int -> int = で，整数と整数から，整数を返す関数
    • +. が float -> float -> float =
  • 数学物理っぽいことをやってた身としては，関数型言語が楽しそうなので，Haskell か lisp を何とかしたいな．
