▽[blog] SNS認知度の低さの理由は？

悲劇的なgooリサーチ結果--SNSは認知も利用もされていない - CNET Japan
SNSの利点は，メンバー全てが誰かを介在して知り合いであると言うことかな． それ故の認知度かもしれない．

SNSの認知度が低いと言うことは，アンケートに回答した人のほとんどが，
  • 「SNSに加入している人を知り合いに持たない」
  • 「SNSに加入している人を知り合いに持つが，紹介してもらっていない」
のどちらかだろう． そもそもSNSってその中身が加入するまでわからないという意味で閉鎖的なんだから， その存在すら気付かない人が多いのは当然だろう． かといって，認知されるように大々的にPRしても， 紹介がなければ加入できないという問題もある．

またSNSとその仕組みは知らないが，mixiなら知ってるとか，livedoorフレンドパークならとか， そういう人は多そう．ただ調査をしたgooリサーチはさすがにmixiなどの名前を出してるだろうから， やっぱり一般的な認知度は低いんだろう． でもまあ，まだSNSが広まり始めてから時間がたってないし， 認知度の上昇には，今後の広がりに期待するしかないかな．

つか認知度が低いにもかかわらず， mixiに入ってる知り合いは凄い多いのは．．．．知り合いが特殊だｋ（ｒｙ．．．．(ノ∀`)

sub getFileList ($) {
    my $dir = $_[0];
    my @files;
    my @unsortedfiles;

    opendir DIR, $dir or die "can't open directory: $!";
    @unsortedfiles = grep /^d\d{8}.hnf$/, readdir DIR;  # Y10K
    closedir DIR;

    @files = sort { $a cmp $b } @unsortedfiles;

    @filelist = ();
    push @filelist, @files;

    foreach $file (@files) {
        $filedir{$file} = $dir;
    }
}
